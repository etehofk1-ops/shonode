연결 사용법

연결 시작 카드 오른쪽 파란 점을 잡고 다음 카드 쪽으로 드래그하세요.

연결 받기 다른 카드 위에 놓거나 왼쪽 점에 꽂으면 그 컷으로 이어집니다.

연결 해제 카드 아래의 `연결 지우기` 버튼으로 분기를 정리할 수 있습니다.

프롬프트 접기 I2I / T2I / I2V 패널 오른쪽 화살표를 눌러 카드 길이를 줄이세요.

빠른 편집 카드를 선택한 뒤 방향키로 위치를 조정하고, `Shift + 방향키`로 더 크게 이동하세요. `Ctrl/⌘ + A`는 전체 선택, `Ctrl/⌘ + D`는 선택 복제입니다.

저장 단축키 `Ctrl/⌘ + S`로 저장하고 `Ctrl/⌘ + Shift + S`로 `.shonode` 파일을 바로 내보낼 수 있습니다.

Workflow

슈노드로 프리비즈와 콘티 흐름을 설계하세요

슈노드는 이미지를 많이 뽑는 툴보다, 브리프에서 컷 구조를 세우고 연결과 프리뷰로 흐름을 검토하는 연출 설계 워크플로입니다.

브리프 → 컷 구조 씬 연결 + 프리뷰 실행 프롬프트 정리
01 브리프에서 연출 구조 시작

브리프, 레퍼런스, 피사체와 룩을 입력하면 슈노드가 컷 흐름과 프롬프트 베이스를 정리합니다.

02 씬과 컷을 연결

컷 목적을 메모하고 연결선을 이어서 장면 흐름과 전환 순서를 한눈에 다듬을 수 있습니다.

03 실행 프롬프트까지 정리

키비주얼, 멀티 앵글, 표정 시트, I2V 프롬프트까지 같은 보드 안에서 정리해 제작 흐름을 이어갑니다.

추천 작업 순서

  1. 브리프와 레퍼런스를 먼저 넣고, 피사체와 룩, 만들고 싶은 영상 목표를 적습니다.
  2. Step 1로 마스터 키비주얼을 만들고 장면의 기준 이미지로 삼습니다.
  3. Step 2와 Step 3으로 멀티 앵글과 표정 시트를 준비해 동일한 정체성을 고정합니다.
  4. Step 4에서 샷 목적, 카메라 감각, 연결 구조를 정리하고 컷 설명을 다듬습니다.
  5. Step 5에서 실제 영상 생성에 넣을 실행 프롬프트를 완성하고 결과 영상을 검토합니다.

편집 팁

  • 컷 설명은 프롬프트보다 장면 목적과 감정선 중심으로 적을수록 다음 생성 단계가 더 안정됩니다.
  • 카드 오른쪽 포트에서 다음 카드로 드래그하면 흐름 연결이 만들어집니다.
  • Ctrl / ⌘ + S로 로컬 저장, Shift를 함께 누르면 `.shonode`로 내보낼 수 있습니다.
  • Ctrl / ⌘ + D는 선택 컷 복제, Delete는 선택 컷 제거입니다.
  • 모바일에서는 하단 탭과 상단 메뉴만으로도 기본 편집 흐름을 이어갈 수 있습니다.

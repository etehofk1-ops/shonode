Shonode Security Quick Scan

링크 하나로
기본 보안 점검

이 페이지는 URL 하나를 넣으면 HTTPS, 보안 헤더, 쿠키 속성, CORS, 혼합 콘텐츠, 간단한 CSRF 힌트까지 빠르게 확인하는 패시브 스캐너입니다. 브라우저 UI는 한 장짜리지만 실제 점검은 서버에서 수행합니다. 더 깊은 액티브 익스플로잇 검증은 Shannon 같은 도구로 이어갈 수 있게 명령도 함께 보여줍니다.

스캔 실행

로컬 실행은 기본 허용입니다. 배포 환경에서는 SHONODE_SECURITY_SCAN_TOKEN 을 맞춰야 스캔이 열립니다. 외부에서 내부망을 스캔하는 SSRF 문제를 막기 위해, 배포 환경에서는 사설망 대상이 기본 차단됩니다.

체크 항목

HTTPS 강제, HSTS, CSP, 클릭재킹 방지, nosniff, Referrer-Policy, Permissions-Policy, 쿠키 플래그, wildcard CORS, target=_blank, mixed content, POST 폼의 CSRF 힌트까지 확인합니다.

결과는 패시브 스캔 기준입니다. "취약"으로 단정하기보다 우선순위와 개선 포인트를 빠르게 정리하는 데 초점을 둡니다.

스캔 결과

대상 URL을 넣고 점검을 시작하면 여기에 상태와 요약이 표시됩니다.
아직 스캔을 실행하지 않았습니다.
